# Отчет по анализу продукта IntraSight Philips в контексте стандартов качества системы ВСУЗИ

## 1. Краткий обзор стандартов ВСУЗИ
Системы внутрисосудистого ультразвукового исследования (ВСУЗИ) должны соответствовать следующим 
критериям качества:
- **Разрешающая способность**: возможность четко различать структуры сосудистой стенки.
- **Глубина визуализации**: способность получать изображение на различных глубинах.
- **Совместимость с катетерами**: возможность использования с различными типами катетеров.
- **Программное обеспечение для анализа**: наличие инструментов для обработки и анализа полученных 
данных.
- **Эргономика**: удобство в использовании и минимизация времени на обучение.
- **Безопасность**: защита данных и соответствие стандартам кибербезопасности.
