# 📂 Prompt Library
Репозиторий содержит структурированную коллекцию оптимизированных промптов для работы с техническими данными, автоматизации бизнес-процессов и анализа технической документации.
И многие другие промпты, используемые в личных и коммерческих целях.

## 🚀 Быстрый старт

Все промпты в библиотеке структурированы по фреймворку **CLEAR** и адаптированы для моделей уровня GPT-4o, Claude 3.5 Sonnet и выше.

### Как использовать:
1. Выберите нужную категорию в папке `prompts/`.
2. Скопируйте содержимое JSON-файла.
3. Используйте переменные в формате `[VARIABLE_NAME]` для подстановки своих данных.

---

## 🛠 Методология разработки
Все промпты проходят через внутренний стандарт качества:
* **Role-based:** Четкое определение экспертной роли.
* **Zero-Hallucination:** Жесткие ограничения на вывод только на основе предоставленного контекста.
* **Structured Output:** Ответы всегда в формате Markdown или JSON для удобной интеграции.

## 📖 Пример структуры промпта (MedTech Audit)

Ниже приведено описание ключевого промпта библиотеки:

* **Owner:** @medtech-engineer
* **Цель:** Анализ полноты и достоверности характеристик медизделий.
